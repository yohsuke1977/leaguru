## 3. 成績の計算入門　https://leaguru.jp/blog/baseball-stats-basics

**①悩み提起**
```
「うちも個人成績つけたい」→でも何を記録して、打率や防御率をどう計算する？で手が止まる人へ。
最低限つけたい項目と計算式を、できるだけシンプルにまとめました。まずはここから🧢 #草野球
https://leaguru.jp/blog/baseball-stats-basics
```

**②気づき**
```
草野球の成績、最初から完璧を目指すと続かない。打撃5項目・投手5項目でOK。
打率＝安打÷打数、出塁率＝(安打+四死球)÷(打数+四死球)。
四死球は打数に含めない、だけ注意。
https://leaguru.jp/blog/baseball-stats-basics
```

**③豆知識**
```
防御率＝自責点×9÷投球回。
でも草野球は7回制も多い。その場合×7で自リーグに合わせる運営者もいる。
大事なのは「リーグ内で計算をそろえる」こと。バラバラだと後でもめる。 #草野球リーグ
https://leaguru.jp/blog/baseball-stats-basics
```

## 4. 試合日程の組み方　https://leaguru.jp/blog/game-schedule-howto

**①悩み提起**
```
シーズン前に必ず通る「試合日程どう組む問題」。
組み方ひとつで終盤の消化試合地獄が決まる。
球場確保→総試合数→ダブルヘッダー→雨天順延、破綻させない順番で整理しました🧢 #草野球
https://leaguru.jp/blog/game-schedule-howto
```

**②気づき/鉄則**
```
日程作りで一番の落とし穴は雨天順延。
「全試合ぴったり収まる」日程は、一度雨が降った瞬間に破綻する。
予備日をシーズン末に確保して、最初から余白をもたせるのが鉄則。
https://leaguru.jp/blog/game-schedule-howto
```

**③豆知識**
```
1回総当たりの試合数＝チーム数×(チーム数−1)÷2。
8チームなら28試合、2回総当たりで56試合。
「開催日数×1日の試合数」で割って収まるか先に確認。足りなきゃ日程段階で破綻確定。 #草野球リーグ
https://leaguru.jp/blog/game-schedule-howto
```

## 5. 会費の決め方　https://leaguru.jp/blog/league-fees-budget

**①悩み提起**
```
草野球リーグの会費、「とりあえず去年と同じ」で決めてない？
それ、年度末に「あれ赤字だ」の典型パターン。
赤字を出さない会費の決め方を、支出の洗い出しから集金のコツまで整理しました🧢 #草野球
https://leaguru.jp/blog/league-fees-budget
```

**②気づき**
```
会費で見落とされがちなのが雨天順延のキャンセル料。
計画段階では存在しないのに、シーズン中は必ず発生する。ここを見込まないだけで赤字に傾く。
予備費は1〜2割の上乗せが安心。
https://leaguru.jp/blog/league-fees-budget
```

**③実感/結論**
```
年会費 vs 参加費どっち？
運営しての結論は「年会費ベース＋球場費だけ実費」の折衷型が一番回しやすい。
予算が読めて、出場数の不公平も小さく、毎試合の集金地獄からも解放される。 #草野球リーグ
https://leaguru.jp/blog/league-fees-budget
```

## 6. トーナメント表の作り方　https://leaguru.jp/blog/tournament-bracket-howto

**①悩み提起**
```
草野球の大会、6チームや12チームだとトーナメント表で急に悩む。
誰をシードに？byeはどう割る？一発勝負でいい？
チーム数が半端なときの組み方まで整理しました🧢 #草野球
https://leaguru.jp/blog/tournament-bracket-howto
```

**②気づき/おすすめ**
```
一発勝負トーナメントの弱点＝1回負けたら朝集まって1試合で終わり。「もう来年いいや」に繋がる。
おすすめは予選リーグ＋決勝トーナメント。全チームが最低複数試合できて満足度が高い。
https://leaguru.jp/blog/tournament-bracket-howto
```

**③豆知識**
```
トーナメントは2のべき乗（4・8・16）できれいに組める。
半端なら近い枠を用意してbyeで埋める（例：6チーム→8枠、上位2チームbye）。
強豪は山を分けて決勝まで当たらないよう散らすのが定石。 #草野球リーグ
https://leaguru.jp/blog/tournament-bracket-howto
```

## 7. エクセル管理はもう限界かも　https://leaguru.jp/blog/excel-standings-limits

**①悩み/あるある**
```
草野球の順位表、エクセルで管理してませんか？
・1試合入力忘れで順位がずれてた
・どれが最新版か分からない
・担当が忙しい週は更新が止まる
開幕は回るけど試合数が増えると必ず崩れる話🧢 #草野球
https://leaguru.jp/blog/excel-standings-limits
```

**②気づき/核心**
```
手作業集計のつらさは「負荷が後半ほど増える」こと。
優勝・タイトル争いで選手が一番順位表を気にする終盤に、集計負荷もピークが来る。
更新が遅れると、せっかくの盛り上がりに水を差す。
https://leaguru.jp/blog/excel-standings-limits
```

**③解決**
```
自動化で取り戻せるのは「時間」と「正確さ」。
試合結果を入力するだけで順位表も打率も自動更新。
運営の仕事が“集計”から“入力”だけに減り、空いた時間をチーム募集や日程調整に回せる。 #草野球リーグ
https://leaguru.jp/blog/excel-standings-limits
```

## 8. 新しいチームが集まるリーグの共通点　https://leaguru.jp/blog/attract-new-teams

**①悩み提起**
```
草野球リーグを続けると必ず来る「チームが足りない」問題。
毎年メンバー集めに苦労するリーグと、宣伝なしで問い合わせが来るリーグ。
その差は「中が外から見えるか」に表れる、という話🧢 #草野球
https://leaguru.jp/blog/attract-new-teams
```

**②気づき**
```
入りたいチームは、連絡する前に必ずリーグを下調べする。
順位表・結果・チーム一覧が公開されてると「土曜開催なら行ける」と前提がそろって連絡が来る。問い合わせの質が上がる。
https://leaguru.jp/blog/attract-new-teams
```

**③核心**
```
新規チーム集めは、派手な宣伝じゃなく「自分で調べて自分で連絡できる導線」を作るだけ。
基本情報・順位表・募集枠・問い合わせ先を一か所に。
入り口が勝手に営業してくれる状態を作る。 #草野球リーグ
https://leaguru.jp/blog/attract-new-teams
```
