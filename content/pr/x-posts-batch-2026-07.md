## 運用メモ

- **ペース**: 1日1本。下の30日カレンダー順に予約投稿すれば、同じ記事が10日以内に重複しない。
- **ハッシュタグ**: #草野球 #草野球リーグ #リーグ運営 等から2〜3個。毎回同じにしない。
- **時間帯**: 草野球層が見る朝（試合前）・夜（帰宅後）・日曜午前が狙い目。
- **併用**: 同じ文面を note のつぶやき／Instagram キャプションにも流用可（IGは外部リンク弱いのでプロフ誘導）。
- **やらないこと**: bp3street 等の募集掲示板に記事を直貼りしない（スパム視される）。掲示板は個別に運営者を見つけて相手のリーグに触れつつ"参考として"渡す用途のみ。
- **反応を見る**: 伸びた切り口（①悩み提起／②気づき／③リスト）を記録し、次バッチはその型に寄せる。

成績の可視化は、選手が「来年も出たい」と思う理由になる。
https://leaguru.jp/blog/why-league-website

### game-schedule-howto ｜ 試合日程の組み方

**① 悩み提起型**
```
シーズン終盤なのに「まだ3試合残ってる」——これ、日程作りの最初でつまずいてます。押さえられる球場と開催日から逆算しないと、優勝争いの最中に日程が詰みます。⚾
#草野球 #日程調整
https://leaguru.jp/blog/game-schedule-howto
```

**② 気づき/エピソード型**
```
日程作りで一番効くのは、実は「雨天の予備日」。ぴったり収まる日程を組むと、一度雨が降った瞬間に破綻します。最初から余白を持たせておく、これに尽きます。📋
#草野球リーグ #運営あるある
https://leaguru.jp/blog/game-schedule-howto
```

**③ リスト/価値型**
```
試合日程、破綻させないコツ。
・開催日×球場数の制約から逆算
・総試合数＝チーム数×(チーム数−1)÷2 を先に把握
・きついならダブルヘッダーも検討
・雨天の予備日は最初に確保
⚾
#草野球 #リーグ運営
https://leaguru.jp/blog/game-schedule-howto
```

### baseball-stats-basics ｜ 個人成績、何を・どう記録

**① 悩み提起型**
```
「そろそろ個人成績つけたい」と思っても、打率の計算式すら怪しくて手が止まる。自責点と失点の違いから分からない…という人、意外と多いはず。
#草野球 #成績管理
https://leaguru.jp/blog/baseball-stats-basics
```

**② 気づき/エピソード型**
```
意外だったのは、四球は打率の分母（打数）に入れないというルール。知らずに計算してる人、草野球だと結構いる。出塁率まで出すと選球眼のいい選手がちゃんと評価されて面白い。
#草野球 #個人成績
https://leaguru.jp/blog/baseball-stats-basics
```

**③ リスト/価値型**
```
個人成績、最初はこれだけでOK。
・打率＝安打÷打数
・出塁率＝(安打＋四死球)÷(打数＋四死球)
・防御率＝自責点×9÷投球回
項目を増やしすぎないのが、記録を続けるコツ。
#草野球 #成績管理
https://leaguru.jp/blog/baseball-stats-basics
```

### league-fees-budget ｜ 年会費・参加費の決め方

**① 悩み提起型**
```
「会費は去年と同じで」で決めていませんか。それ、年度末に「あれ、赤字だ」ってなる典型パターンです。運営費は支出から逆算しないと、後でじわじわ足が出ます。📋
#草野球 #リーグ運営
https://leaguru.jp/blog/league-fees-budget
```

**② 気づき/エピソード型**
```
会費設計で一番見落とされがちなの、実は「雨天順延のキャンセル料」。計画段階には存在しないのに、シーズン中は必ず発生します。ここを予備費に入れておくかで収支が変わります。⚾
#草野球リーグ #運営あるある
https://leaguru.jp/blog/league-fees-budget
```

**③ リスト/価値型**
```
赤字にしない会費の決め方。
・支出を洗い出す（予備費込み）
・固定費＋変動費を出す
・1〜2割の予備費を乗せる
・チーム数で割る
「年会費＋球場費だけ実費」の折衷型が一番回しやすいです。📋
#草野球 #リーグ運営 #会計
https://leaguru.jp/blog/league-fees-budget
```
