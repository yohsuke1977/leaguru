# 記事 → SNS一括展開の型（全プロダクト共通）

ブログ記事1本を、X（単発＋連投）・note・Instagram・個別DM用に切り出すための再利用テンプレ。
**書く（オンページ）は足りている。撒く（配信）が本体。** 記事を書いたら必ずこの型で撒く。

- 対象プロダクト: Leaguru / Ninepage / taxi-log / meru / 推し通知 / driver-tenshoku / machi-manga 等どれでも可
- 使い方: 下の「再利用プロンプト」に記事本文と変数を入れてClaude/Fableに投げる → 出力を予約投稿にまとめ録り

---

## 再利用プロンプト（コピペして使う）

```
あなたはSNS運用の担当者。以下のブログ記事を、各媒体向けの投稿文に切り出してください。
売り込み臭を消し、「運営/現場のリアルな実感」トーンで。絵文字は控えめ（1投稿1〜2個）。

# 入力
- プロダクト名: {{例: Leaguru}}
- 一言説明: {{例: 草野球リーグ運営SaaS・年額¥18,000}}
- ターゲット読者: {{例: 草野球リーグの運営者・幹事}}
- 記事タイトル: {{}}
- 記事URL(CTA): {{例: https://leaguru.jp/blog/xxx}}
- 記事本文: 
{{ここに本文を貼る}}

# 出力（この順で）
1. X単発 ×3（切り口を変える：①悩み提起型 ②気づき/エピソード型 ③リスト/価値型）。各140字以内、末尾にURL。
2. X連投（4〜6ツイート）。1ツイート目はフックのみ、最後にURL。各140字以内。
3. note用リード文（記事を転載する際の冒頭2〜3行の個人的な導入＋末尾CTA）。
4. Instagram画像カード用テキスト（見出し1＋箇条書き3〜4）。※画像に載せる想定で短く。
5. 配信時メモ（この記事を撒くのに向いた生息地・避けるべき場所）。
```

## 配信ルール（媒体別の register）

| 媒体 | やり方 | 注意 |
|---|---|---|
| **X 単発** | 切り口を変えて別日に複数回投稿（使い回しOK） | ハッシュタグは #草野球 #草野球リーグ 等2〜3個まで |
| **X 連投** | 記事の要約スレッド。最後にだけURL | 1本目でフック（結論/意外性）を出す |
| **note** | 記事をほぼ転載＋個人的リード。noteは検索/おすすめ流入がある | canonicalは自社ブログ優先（重複対策）。noteは"入口"と割り切る |
| **Instagram** | 画像カード（machi-mangaの合成技術で作れる）＋キャプションにリンク誘導 | IGは外部リンクが弱い→プロフ誘導 or ハイライト |
| **掲示板（bp3street等）** | ❌ 記事の広告貼りはNG（スパム扱い） | 掲示板は「参加募集」の場。使うなら運営者を見つけて**個別に**、相手のリーグに触れつつ記事を"参考として"渡す |
| **個別DM/メール** | ⭐ ikz.jp等で自前運営してる運営者に、記事を"役立つ情報"として送る | 一斉送信しない。1件ずつ相手のリーグ名を入れる |

## 実出力サンプル：記事「なぜ草野球リーグに公式サイトが必要なのか」

URL: https://leaguru.jp/blog/why-league-website ／ 対象: リーグ運営者

### 1. X単発 ×3

**① 悩み提起型**
```
「うちのリーグ、公式サイト無いよ」——草野球だとこれが普通。
でも順位表が運営者のエクセルの中だけにあると、
・今何位？が本人しか分からない
・成績が残らない
・新規チームが不安で入りづらい
サイト1つで全部変わる話を書きました🧢
https://leaguru.jp/blog/why-league-website
```

**② 気づき/エピソード型**
```
リーグ運営してて一番驚いたのは、“自分の打率がサイトに残る”だけで選手の熱量が変わること。
草野球は趣味。それでも「今年は3割超えたい」が最後の試合を本気にさせる。
成績の可視化は、選手が「来年も出たい」と思う理由になる。
https://leaguru.jp/blog/why-league-website
```

**③ リスト/価値型**
```
草野球リーグに公式サイトがあると効くこと4つ👇
①順位表を誰でもスマホで見られる
②成績が残り選手のモチベに
③新規チームが安心して問い合わせできる
④「あれどこ？」の連絡コストが激減
“無くても回る”けど“あると確実に良くなる”。
https://leaguru.jp/blog/why-league-website
```

### 2. X連投（5ツイート・1枠＝1ツイート）

```
1/ 「草野球リーグに公式サイトって要る？」
結論、無くても回る。でも“あると確実に良くなる”。
リーグ運営に関わって実感したことをまとめます🧵
```

```
2/ 公式サイトが無いと、リーグの情報が“運営者の頭とエクセル”に閉じ込められる。
・今何位？が本人しか分からない
・先週の結果を他チームが知れない
・自分の記録が残らない
```

```
3/ サイトがあれば順位表が自動更新でスマホから誰でも見られる。
「今何位？」に毎回答える手間が消えて、試合の無い週末もみんながリーグを追える。
```

```
4/ 一番効くのは成績。打率やHRが残るだけで「今年は3割」「あいつより打点を」。
趣味なのに本気になる。これが“来年も出たい”に繋がる。
```

```
5/ 新規チームもネットで調べてから連絡する時代。
ちゃんと運営されてると一目で分かれば問い合わせのハードルが下がる。
運営者向けに詳しく書きました👇
https://leaguru.jp/blog/why-league-website
```

### 3. note用リード文

```
草野球のリーグ運営に関わってきて、「公式サイトがあるリーグと無いリーグって、選手の熱量も新規チームの集まり方も全然違うな」と感じています。
「無くても回る」は事実。でも「あると確実に良くなる」。その理由を、現場の実感ベースで整理しました。

（…本文転載…）

▼リーグ運営に必要な機能を一つにまとめたサービスを作っています： https://leaguru.jp
```

### 4. Instagram画像カード用テキスト

```
草野球リーグに公式サイトが必要な4つの理由
・順位表を誰でもスマホで
・成績が残る＝選手のモチベ
・新規チームが安心して入れる
・連絡コストが激減
```
（キャプション末尾で「詳しくはプロフのリンクから」）

### 5. 配信メモ

- ⭕ X（#草野球 #草野球リーグ）／note／自前運営リーグへの個別DM
- ⭕ Instagram：machi-mangaの画像合成でカード量産できる（横展開しやすい）
- ❌ bp3street等の募集掲示板に直貼りしない（スパム視される）
